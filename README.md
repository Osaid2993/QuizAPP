# QuizAPP

## About the project
This is an Android quiz app made for SIT305 Credit Task 3.1. The app lets users enter their name, attempt a multiple choice quiz, get visual feedback on their answers, and view their final score.

## Features
- Enter your name to start the quiz
- Answer 5 multiple choice questions based on SIT305 unit content
- Submit answers and get instant colour feedback
- Progress bar updates as you move through the quiz
- View your final score on a results screen
- Retake the quiz without re-entering your name
- Close the app from the results screen

## How the quiz works
- Correct answer turns green
- Wrong answer turns red, correct one also highlighted green
- Answers lock after submission
- Score is tracked across all questions

## Technologies used
- Java
- XML
- Android Studio
- Material3 Theme

## How to run the app
1. Open the project in Android Studio
2. Let Gradle sync
3. Run the app on an emulator or Android device
